# WebNovel - Fullstack Starter

This is a starter Next.js + Prisma scaffold for the WebNovel project.

Quickstart (local):
1. Install Node.js >=18
2. `npm install`
3. Use SQLite for quick demo: ensure `prisma/schema.prisma` provider is set to sqlite
4. `npx prisma migrate dev --name init`
5. `npm run dev`

For Vercel deploy: push this repo to GitHub and import into Vercel.
