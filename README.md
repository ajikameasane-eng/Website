# WebNovel - Frontend-only (GitHub Pages)
This is a single-file demo app that runs entirely in the browser (localStorage).
Save `index.html` and open in a browser or host via GitHub Pages.

Features:
- Register / login (local demo)
- Create story, volumes, chapters
- Upload images (stored as base64 in localStorage)
- Follow, comment, report (local-only)

Notes:
- Not suitable for production. For real deployment, use the `webnovel-fullstack` folder.
